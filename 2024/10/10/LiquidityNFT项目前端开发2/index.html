Ant Design（Antd）栅格系统

Ant Design（Antd）的栅格系统基于 Flexbox 布局，用于创建响应式布局。它的设计目标是帮助开发者在不同的屏幕尺寸上创建一致的布局结构。栅格系统提供了非常灵活的方式来创建响应式布局，并基于 Flexbox 提供了强大的对齐和布局能力。你可以利用 Row 和 Col 的属性来根据屏幕大小和内容需求轻松调整布局。

1. 栅格的基本概念

Antd 的栅格系统主要由两部分组成：

  • Row（行）：用来创建行，它在水平方向上容纳 Col。
  • Col（列）：用来创建列，它在水平和垂直方向上对齐内容，并定义其在栅格中的宽度。

2. 栅格的属性

Row 属性

  • gutter: 设置栅格之间的间距。可以是单个数字（像素），也可以是数组（分别表示水平方向和垂直方向的间距）。

    或者设置水平方向和垂直方向不同的间距：

  • align: 设置列的垂直对齐方式。可选值为 top，middle，bottom。

  • justify: 设置列的水平排列方式。可选值为 start，end，center，space-around，space-between。

Col 属性

  • span: 设置列在栅格系统中的占比，最大值为 24。

  • offset: 设置列的左侧偏移，最大值为 24。

  • push / pull: 用于移动列的位置，push 向右移动，pull 向左移动。

3. 响应式布局

Antd 的栅格系统支持响应式布局，允许根据不同的屏幕尺寸动态调整列宽。支持的断点如下：

  • xs：<576px
  • sm：≥576px
  • md：≥768px
  • lg：≥992px
  • xl：≥1200px
  • xxl：≥1600px

4. 使用 Flex 布局

Antd 的栅格基于 Flexbox，因此可以直接利用 Flex 的特性对内容进行布局。

  • 可以通过 justify 属性来控制水平对齐方式，例如居中对齐或两端对齐。
  • 通过 align 来控制垂直方向的对齐。
